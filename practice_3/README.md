# Практическое занятие 3

Тема: технологии экранного доступа в ОС Linux (Orca) 

## 1. Создание текстового файла

После скичивания ORCA командой sudo apt install orca, он был сочетанием клавиш Alt+Super+S.

Текстовый файл с именем P4150_Badmaev.txt был создан и отредактирован, шаги:

* touch P4150_Badmaev.txt
* nano P4150_Badmaev.txt
* вписал текст и сохранил

## 2. Отправка письма с вложением

Для отправки письма использовался браузер Chromium. Трудностей не было, все также как и на windows.
Единственное, когда я зашел на почту ЯНДЕКС, вылез огромный баннер с рекламой на весь экран, немного растерялся.
Проблема решилась просто кнопкой "ESC"

Шаги выполнения:

1. Открыт сайт почтового сервиса по адресу электронной почты.
2. С помощью клавиши Tab произведена навигация между элементами страницы («Написать письмо», поля ввода, кнопки).
3. В форме нового письма заполнены поля «Получатель» — *[nikkov95@mail.ru](mailto:nikkov95@mail.ru)*, тема и текст сообщения.
4. Через кнопку «Прикрепить файл» выбран созданный ранее текстовый файл.
5. Кнопкой «Отправить» письмо было отправлено.
