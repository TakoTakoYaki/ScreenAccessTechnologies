## 1. Краткое описание проекта

В рамках задания создан небольшой статический сайт на тему волейбола.  
Главная цель — реализовать сайт, полностью **недоступный** для программ экранного доступа.  
Для этого весь контент помещён в контейнер с атрибутом `aria-hidden="true"`, что скрывает его для скринридеров.

---

## 2. Структура сайта

Сайт состоит из трёх простых страниц:

1. **index.html** — главная, краткое описание игры.  
2. **positions.html** — основные позиции игроков в волейболе.  
3. **rules.html** — базовые правила игры.
